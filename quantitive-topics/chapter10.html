Chapter 10 - Mathematical Reasoning & Sets

1. Venn Diagrams & Set Theory

Set Theory

  • Set Theory deals with the study of sets, which are collections of objects or elements.
  • For example, the set of all natural numbers, the set of all students in a class, etc.

Basic Terminology in Set Theory

  • Set: A collection of well-defined objects. For example, A = {1, 2, 3} is a set containing the numbers 1, 2, and 3.
  • Element: An individual object in a set. For example, 1 is an element of the set A = {1, 2, 3}.
  • Subset: A set where all its elements are also contained in another set. For example, A = {1, 2} is a subset of B = {1, 2, 3}.
  • Union (A ∪ B): The set of all elements that belong to either set A, set B, or both.
  • Intersection (A ∩ B): The set of all elements that belong to both set A and set B.
  • Difference (A − B): The set of all elements that belong to A but not to B.
  • Complement (Aᶜ): The set of all elements not in A.
  • Universal Set (U): The set that contains all possible elements under consideration.

Example of Set Operations

  • A = {1, 2, 3} and B = {2, 3, 4}
  • Union: A ∪ B = {1, 2, 3, 4}
  • Intersection: A ∩ B = {2, 3}
  • Difference: A − B = {1}
  • Complement: If the universal set U = {1, 2, 3, 4, 5}, then Aᶜ = {4, 5}

Venn Diagrams

  • A Venn Diagram is a visual representation of sets and their relationships with each other.
  • It uses overlapping circles to show how sets relate to one another in terms of union, intersection, and difference.

Venn Diagram Components

  • Universal Set (U): Represented by a rectangle, it includes all possible elements under consideration.
  • Circles: Represent sets within the universal set. Overlapping circles show the intersection of sets.
  • Shaded Regions: Used to illustrate operations like union, intersection, and difference.

Operations in Venn Diagrams

  • Union: The area covered by both sets. If you have two sets A and B, the union is the entire area covered by both circles.
  • Intersection: The area where both circles overlap. It shows the common elements between two sets.
  • Difference: The area in one circle that does not overlap with the other circle. It shows the elements that belong to one set but not the other.
  • Complement: The area outside a circle represents the complement of the set. It includes all elements that do not belong to the set.

Types of Venn Diagrams

  1. Two-Set Venn Diagram

    • A two-set Venn diagram is used to represent the relationships between two sets.
    • Example: A = {1, 2, 3}, B = {2, 3, 4}
    • Union: A ∪ B = {1, 2, 3, 4}
    • Intersection: A ∩ B = {2, 3}

  2. Three-Set Venn Diagram

    • A three-set Venn diagram is used to represent the relationships between three sets.
    • Example: A = {1, 2, 3}, B = {2, 3, 4}, C = {3, 4, 5}
    • Union: A ∪ B ∪ C = {1, 2, 3, 4, 5}
    • Intersection: A ∩ B ∩ C = {3}

Examples of Venn Diagram Problems

Example 1:

In a survey, 80 people were asked if they liked tea or coffee. 50 liked tea, 40 liked coffee, and 30 liked both tea and coffee. How many liked only tea, only coffee, or both?

  • T = Tea lovers, C = Coffee lovers
  • ∣T∣ = 50 (Tea lovers)
  • ∣C∣ = 40 (Coffee lovers)
  • ∣T ∩ C∣ = 30 (People who like both)
  • Only Tea = ∣T∣ − ∣T ∩ C∣ = 50 − 30 = 20
  • Only Coffee = ∣C∣ − ∣T ∩ C∣ = 40 − 30 = 10
  • Thus, the final answers are:
    • 20 people like only tea.
    • 10 people like only coffee.
    • 30 people like both tea and coffee.

Example 2:

In a class of 60 students, 40 students like Mathematics, 35 like Physics, and 25 like both Mathematics and Physics. How many students like only one of the subjects?

  • M = Students who like Mathematics, P = Students who like Physics
  • ∣M∣ = 40
  • ∣P∣ = 35
  • ∣M ∩ P∣ = 25
  • Only Mathematics = ∣M∣ − ∣M ∩ P∣ = 40 − 25 = 15
  • Only Physics = ∣P∣ − ∣M ∩ P∣ = 35 − 25 = 10
  • Thus, the number of students who like only one subject is:
    • 15 students like only Mathematics.
    • 10 students like only Physics.

2. Logical Deductions

Definition

  • Logical deduction refers to the process of drawing conclusions based on given facts, rules, or premises.
  • It is a critical skill in many areas such as mathematics, philosophy, and competitive exams.
  • In logical deduction problems, you are provided with a set of conditions, and your task is to infer the correct relationships or conclusions from those conditions.

Types of Logical Deduction Problems

Syllogism

  • Syllogism is a type of logical argument where a conclusion is drawn from two given or assumed propositions (premises).
  • Example:
    • Premise 1: All men are mortal.
    • Premise 2: Socrates is a man.
    • Conclusion: Socrates is mortal.

    • Syllogism problems typically involve two premises, and you need to deduce the correct conclusion based on the given facts.

    Example Problem:

    • Premise 1: All dogs are animals.
    • Premise 2: Fido is a dog.
    • Conclusion: Fido is an animal.
    • Answer: The conclusion is valid, as it follows from the premises.

Puzzle-based Logical Deductions

  • These puzzles involve a set of conditions that describe relationships between different items or entities. You need to use these conditions to deduce certain facts or answer questions.

    • Example Problem:

    There are four friends: A, B, C, and D.

    Conditions:

    • A is taller than B.
    • C is taller than D.
    • B is shorter than D.

    Question:

    Who is the tallest among the four friends?

    Solution:

    • A > B (A is taller than B).
    • C > D (C is taller than D).
    • B < D (B is shorter than D).
    • Since A is taller than B, and B is shorter than D, we conclude that C is the tallest, as there is no restriction on C's height relative to A.
    • Answer: C is the tallest.

Direction and Position-based Problems

  • These problems involve determining someone's location, position, or direction based on a series of statements.

    • Example Problem:

    A person walks 10 meters North, turns right, and walks 10 meters again. Then, he turns right and walks 10 meters. In which direction is he from his starting point?

    Solution:

    • The person walks 10 meters North.
    • Then, he turns right (East) and walks 10 meters.
    • Finally, he turns right again (South) and walks 10 meters.
    • Since the total vertical displacement (North-South) is 0 (10 meters North and 10 meters South), the person is directly East of the starting point.
    • Answer: The person is to the East of the starting point.

Odd One Out

  • These problems ask you to identify an item that doesn't belong in a given set or group.

    • Example Problem:

    • A. Lion
    • B. Tiger
    • C. Bear
    • D. Leopard

    Solution:

    • The first four animals (Lion, Tiger, and Leopard) are all big cats, while the Bear is not a big cat.
    • Answer: Bear is the odd one out.

Blood Relations

  • Logical deduction can be applied to blood relation problems, where you need to deduce the relationship between family members based on given information.

    • Example Problem:

    A woman is looking at a photograph. A person asks her, "Who are you looking at?" The woman replies, "Brothers and sisters, I have none. But the father of the person in the photograph is my father’s son." Who is in the photograph?

    Solution:

    • The woman says, "The father of the person in the photograph is my father’s son." The person in the photograph is her son because her father’s son would be her brother, and the father of the person in the photograph would then be her son.
    • Answer: The person in the photograph is the woman's son.

Statements and Assumptions

  • Logical deduction can also involve analyzing a statement and inferring whether the assumption based on it is true or false.

    • Example Problem:

    Statement: "No birds are mammals." Assumption: "All birds are non-mammals."

    Solution:

    • The statement is saying that no birds are mammals, which implies that all birds are non-mammals. Thus, the assumption is true.
    • Answer: The assumption is correct.

Approaching Logical Deduction Problems

  • Read the problem carefully: Understand all the conditions or premises provided.
  • Break down the problem: Identify key facts and relationships that are mentioned.
  • Use process of elimination: In some cases, you can eliminate incorrect options and deduce the correct answer.
  • Draw diagrams if necessary: Sometimes, drawing a Venn diagram, family tree, or chart helps visualize relationships and makes deductions easier.
  • Consider all possibilities: When you’re unsure about a problem, list all possible scenarios and narrow them down based on the given conditions.

Practice Problems

Syllogism

  • Premise 1: All roses are flowers.
  • Premise 2: Some flowers are red.
  • Conclusion: Some roses are red. Is this conclusion valid?

Puzzle

There are five friends: A, B, C, D, and E. They are sitting in a row. C is sitting to the immediate right of B. A is sitting to the immediate left of D. E is sitting to the immediate left of C.

Question:

Who is sitting in the middle?

Blood Relation

3. Mathematical Puzzles & Riddles

Introduction

Mathematical puzzles and riddles are fun and challenging ways to test your problem-solving skills and enhance your logical thinking. They often require creative thinking and an understanding of basic mathematical concepts. These puzzles can range from simple arithmetic to complex logic-based problems.

Common Types of Mathematical Puzzles and Riddles

1. Classic Number Puzzles

Puzzle 1: The Missing Dollar Riddle

Problem: Three friends check into a hotel room that costs $30. They each contribute $10, and the hotel clerk gives them a $5 discount because of an overcharge. The clerk gives $1 back to each friend and keeps $2 for himself. Now, each friend has paid $9 (totaling $27), and the clerk has kept $2. Where did the extra dollar go?

Solution:

This puzzle is often misleading because it suggests a discrepancy in the way the amounts are being added.

  • The total amount paid is $30.
  • Each friend gets $1 back, so they have paid $9 each, totaling $27.
  • Out of this $27, $25 is for the room, and $2 is kept by the clerk.
  • There is no missing dollar; the confusion comes from adding the $2 (the clerk's tip) to the $27, which is incorrect. The $27 already includes the $2 kept by the clerk.

Answer: The riddle plays on a logical error in the way the amounts are added.

2. Word-based Mathematical Riddles

Puzzle 2: The Bridge and Torch Problem

Problem: Four people need to cross a bridge at night. They have one torch, and the bridge can only hold two people at a time. The crossing times are:

  • Person 1: 1 minute
  • Person 2: 2 minutes
  • Person 3: 5 minutes
  • Person 4: 10 minutes

The torch must be walked back each time, and two people can cross together, but they must move at the slower person's pace. What is the shortest time in which all four people can cross the bridge?

Solution:

  • First crossing: Person 1 and Person 2 cross together (2 minutes).
  • Return: Person 1 goes back with the torch (1 minute).
  • Second crossing: Person 3 and Person 4 cross together (10 minutes).
  • Return: Person 2 goes back with the torch (2 minutes).
  • Final crossing: Person 1 and Person 2 cross together again (2 minutes).

Total time = 2 + 1 + 10 + 2 + 2 = 17 minutes.

Answer: The shortest time is 17 minutes.

3. Logic-based Mathematical Riddles

Puzzle 3: The Two Sons Riddle

Problem: A man is looking at a picture of someone. His friend asks, “Who are you looking at?” The man replies, “Brothers and sisters, I have none. But the father of the person in the picture is my father’s son.” Who is in the picture?

Solution:

The man says, "The father of the person in the picture is my father’s son."

  • If the man has no brothers, "my father’s son" must refer to himself.
  • Therefore, the person in the picture is his son.

Answer: The person in the picture is his son.

4. Age-related Puzzles

Puzzle 4: The Age Difference Problem

Problem: A father is 30 years older than his son. In 10 years, the father will be three times as old as the son. How old are the father and the son?

Solution:

Let the son's age be x. Then the father's age is x + 30.

In 10 years, the son will be x + 10, and the father will be x + 40.

According to the problem, in 10 years, the father will be three times the son's age:

x + 40 = 3(x + 10)

Simplifying:

x + 40 = 3x + 30

40 - 30 = 3x - x

10 = 2x

x = 5

So, the son is 5 years old, and the father is 5 + 30 = 35 years old.

Answer: The son is 5 years old, and the father is 35 years old.

5. Pattern and Sequence Puzzles

Puzzle 5: The Number Sequence

Problem: What is the next number in the sequence: 1, 4, 9, 16, 25, __?

Solution:

The numbers are perfect squares:

  • 12 = 1
  • 22 = 4
  • 32 = 9
  • 42 = 16
  • 52 = 25

The next number in the sequence will be 62 = 36.

Answer: The next number is 36.

6. Coin-related Puzzle

Puzzle 6: The Weighing Puzzle

Problem: You have 12 identical-looking coins. One of them is either heavier or lighter than the others. You have a balance scale and can only use it three times. How do you find the odd coin and determine whether it is heavier or lighter?

Solution:

  • Divide the 12 coins into three groups of 4 coins each.
  • Weigh two groups (Group A and Group B). There are three possible outcomes:
    • If they balance, the odd coin is in Group C.
    • If Group A is heavier or lighter, the odd coin is in Group A or Group B.
  • Now take the 4 coins from the group that contains the odd coin and divide them into three coins and one coin.
  • Weigh the three coins against each other:
    • If they balance, the odd coin is the one not weighed, and you can determine whether it is heavier or lighter.
    • If they do not balance, you will know which coin is the odd one and whether it is heavier or lighter.

Answer: This method will find the odd coin and determine if it is heavier or lighter in three weighings.

7. Logical Deduction Puzzle

Puzzle 7: The Light Bulb Problem

Problem: You are in a room with three switches, each controlling one of three light bulbs in another room. You can’t see the bulbs from where the switches are, and you can only enter the room with the bulbs once. How do you determine which switch controls which bulb?

Solution:

  • Turn on the first switch and leave it on for a few minutes.
  • After a few minutes, turn off the first switch and turn on the second switch.
  • Now, enter the room with the bulbs:
    • The bulb that is on corresponds to the second switch.
    • The bulb that is off and warm corresponds to the first switch (since it was on for a while and then turned off).
    • The bulb that is off and cold corresponds to the third switch.

4. Statements & Assumptions

Definition

  • A statement is a sentence that conveys some information.
  • An assumption is something taken as true for the sake of argument, even if it may not be explicitly stated.
  • The goal in problems involving statements and assumptions is to determine whether an assumption logically follows from the statement provided.

Rules for Assumptions

  • An assumption is valid if it is something that is implied by the statement, even if not explicitly mentioned.
  • An assumption is invalid if it goes beyond the information provided or introduces something that contradicts the statement.

Types of Assumptions

  1. Implicit Assumptions

    • Assumptions logically derived from the given statement.

  2. Explicit Assumptions

    • Assumptions that are directly or indirectly mentioned in the statement.

Example Problems

  1. Example 1: Statement: "This project will be completed in two months."

    • Assumption 1: The project will be completed within the given timeframe. (Valid)
    • Assumption 2: The resources required for the project are available. (Invalid)

  2. Example 2: Statement: "We have enough time to complete the task, so we are not in a hurry."

    • Assumption 1: The task will take longer to complete than expected. (Invalid)
    • Assumption 2: The task can be completed without rushing. (Valid)

  3. Example 3: Statement: "The weather will be great for the outdoor event tomorrow."

    • Assumption 1: The weather is expected to be good. (Valid)
    • Assumption 2: The outdoor event will take place tomorrow without any problems. (Invalid)

  4. Example 4: Statement: "All workers must report to the office at 9 AM tomorrow for the meeting."

    • Assumption 1: All workers are expected to attend the meeting. (Valid)
    • Assumption 2: The meeting is scheduled for tomorrow at 9 AM. (Valid)

  5. Example 5: Statement: "The company will provide training for all new employees starting next week."

    • Assumption 1: All new employees will be trained. (Valid)
    • Assumption 2: The training will begin next week. (Valid)

  6. Example 6: Statement: "The new law will make driving safer for everyone."

    • Assumption 1: The new law will improve driving safety. (Valid)
    • Assumption 2: Everyone will follow the new law. (Invalid)

  7. Example 7: Statement: "The car has been in the shop for a week, and it's still not ready."

    • Assumption 1: The car is undergoing repairs. (Valid)
    • Assumption 2: The car will be ready in the future. (Valid)

  8. Example 8: Statement: "The company plans to expand its operations internationally within the next five years."

    • Assumption 1: The company will open new branches in other countries. (Valid)
    • Assumption 2: The company will have the necessary resources for international expansion. (Invalid)

  9. Example 9: Statement: "I will be attending the party tonight."

    • Assumption 1: The party is happening tonight. (Valid)
    • Assumption 2: The person has been invited to the party. (Invalid)
