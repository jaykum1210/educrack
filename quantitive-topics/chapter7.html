Chapter 7 - Probability & Statistics

Basic Probability Concepts

Probability is a branch of mathematics that deals with calculating the likelihood or chance of an event occurring. It is widely used in various fields such as statistics, finance, gaming, and science.

i. What is Probability?

Probability is the measure of how likely an event is to occur, expressed as a number between 0 and 1.

The probability of an event (P(E)) is given by:

P(E) = Number of favorable outcomes / Total number of possible outcomes

where:

  • P(E) is the probability of event E.
  • The numerator is the number of favorable outcomes (the outcomes we are interested in).
  • The denominator is the total number of possible outcomes (all outcomes that could possibly occur).

ii. Types of Events

Certain Event:

An event that is guaranteed to occur.

Example: The probability of rolling a number between 1 and 6 on a standard die is 1 (i.e., P(Rolling 1-6) = 1).

Impossible Event:

An event that cannot occur.

Example: The probability of rolling a 7 on a standard 6-sided die is 0 (i.e., P(Rolling 7) = 0).

Simple Event:

An event that has only one outcome.

Example: Rolling a 4 on a standard die.

Compound Event:

An event that involves two or more simple events happening together.

Example: Rolling an even number (i.e., 2, 4, or 6) on a standard die.

iii. Basic Probability Rules

The Probability of the Sample Space:

The probability of the entire sample space (all possible outcomes) is always 1.

P(Sample Space) = 1

The Probability of Complementary Events:

The probability of the complement of an event A (i.e., the event that A does not occur) is given by: 

P(A') = 1 - P(A)

where P(A') is the probability of event A not happening.

Addition Rule:

The addition rule is used to find the probability of the occurrence of either of two events. For two events A and B:

  • For mutually exclusive events (events that cannot happen at the same time):
    P(A ∪ B) = P(A) + P(B)
  • For non-mutually exclusive events (events that can happen at the same time):
    P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Multiplication Rule:

The multiplication rule is used to find the probability of two independent events happening together:

  • For independent events (events that do not affect each other):
    P(A ∩ B) = P(A) × P(B)
  • For dependent events (events where the occurrence of one affects the occurrence of the other):
    P(A ∩ B) = P(A) × P(B|A)

where P(B|A) is the conditional probability of B occurring given that A has already occurred.

iv. Conditional Probability

Conditional probability is the probability of an event occurring given that another event has already occurred. It is represented as P(A|B), the probability of event A occurring given event B.

The formula for conditional probability is:

P(A|B) = P(A ∩ B) / P(B)

where:

  • P(A ∩ B) is the probability that both A and B occur.
  • P(B) is the probability that event B occurs.

v. Sample Space and Events

The sample space (denoted S) is the set of all possible outcomes of a random experiment. An event is any subset of the sample space.

Example 1: Rolling a Die

Sample space S = {1, 2, 3, 4, 5, 6}

Event A: Rolling an even number A = {2, 4, 6}

Event B: Rolling a number greater than 4 B = {5, 6}

The probability of event A is:

P(A) = Number of favorable outcomes for A / Total number of outcomes = 3 / 6 = 0.5

Example 2: Tossing a Coin

Sample space S = {H, T} (H for heads, T for tails)

Event A: Getting heads A = {H}

The probability of event A is:

P(A) = 1 / 2 = 0.5

vi. Types of Probability

Theoretical Probability:

This is based on the assumption that all outcomes are equally likely. For example, when tossing a fair coin, the probability of getting heads is 1/2 because there are two equally likely outcomes: heads or tails.

Experimental Probability:

This is based on observed outcomes of an experiment. It is calculated as the ratio of the number of times an event occurs to the total number of trials.

P(E) = Number of favorable outcomes / Total number of trials

For example, if a die is rolled 100 times and the number 6 appears 20 times, the experimental probability of rolling a 6 is 20/100 = 0.2.

Empirical Probability:

Similar to experimental probability, but typically based on a larger set of data or experiments.

vii. Law of Large Numbers

The Law of Large Numbers states that as the number of trials or experiments increases, the experimental probability of an event will approach the theoretical probability. For example, if you flip a fair coin many times, the proportion of heads should approach 0.5 as the number of flips increases.

viii. Probability Distribution

A probability distribution is a function that gives the probability of each outcome in a sample space. There are two main types of probability distributions:

  • Discrete Probability Distribution: For events with a finite number of possible outcomes (e.g., rolling a die).
  • Continuous Probability Distribution: For events with an infinite number of possible outcomes (e.g., the probability of selecting a number from a continuous range).

2. Mean, Median, and Mode

Definition

  • Mean, Median, and Mode are measures of central tendency that summarize a dataset by identifying a "typical" value.

Mean

  • The **mean** (average) is calculated by summing all values and dividing by the total number of values.
  • Formula: Mean = ΣX n

    where ΣX = sum of all values, n = number of values.

  • Example: Dataset = 5, 8, 12, 14, 20
  • Mean = (5 + 8 + 12 + 14 + 20) / 5 = 59 / 5 = 11.8

SI Unit

  • The unit of mean is the same as the unit of the data (e.g., meters, seconds, etc.).

Median

  • The **median** is the middle value in an ordered dataset.
  • If the number of values is odd, the median is the middle value; if even, it's the average of the two middle values.
  • Example (Odd number of values): Dataset = 7, 2, 9, 3, 5. Ordered = 2, 3, 5, 7, 9. Median = 5.
  • Example (Even number of values): Dataset = 4, 2, 8, 10, 6, 12. Ordered = 2, 4, 6, 8, 10, 12. Median = (6 + 8) / 2 = 7.

Mode

  • The **mode** is the value that appears most frequently in a dataset.
  • Example (No mode): Dataset = 2, 3, 4, 5, 6 (No repetitions, no mode).
  • Example (One mode): Dataset = 2, 3, 3, 5, 6 (Mode = 3).
  • Example (Two modes): Dataset = 1, 2, 3, 3, 4, 5, 5 (Mode = 3, 5).
  • Example (Multiple modes): Dataset = 1, 1, 2, 2, 3, 3, 4, 4 (Mode = 1, 2, 3, 4).

Comparison of Mean, Median, and Mode

Measure Definition Effect of Outliers Best Used For
Mean Average of all values Affected by outliers Symmetrical (normal) distribution
Median Middle value of ordered data Not affected by outliers Skewed data or data with outliers
Mode Most frequent value Not affected by outliers Categorical data or identifying the most common value

Applications of Mean, Median, and Mode

  • **Mean**: Used to calculate the average score in exams, wages, etc.
  • **Median**: Used in income distribution, housing prices, and other data where outliers can skew results.
  • **Mode**: Used in market research to identify the most popular product or service.

3. Variance and Standard Deviation

Definition

  • Variance and Standard Deviation are measures of spread or dispersion in a dataset.
  • They describe how much the data points in a dataset differ from the mean (average) of the dataset.

Variance

  • The **variance** measures the average squared deviation of each data point from the mean.
  • Formula for **Population Variance**: Variance ( σ ² ) = Σ(Xi - μ)2 N

    Where σ² = population variance, Xi = each data point, μ = population mean, N = total number of data points.

  • Formula for **Sample Variance**: Variance ( s ² ) = Σ(Xi - X̄)2 n - 1

    Where s² = sample variance, Xi = each data point, X̄ = sample mean, n = total number of data points, and (n-1) is for Bessel's correction.

  • Steps to Calculate Variance:
    • Find the mean of the dataset.
    • Subtract the mean from each data point and square the result.
    • Find the average of these squared deviations.
  • Example (Population Variance):
    • Dataset = 2, 4, 6, 8, 10
    • Mean (μ) = (2 + 4 + 6 + 8 + 10) / 5 = 6
    • Squared deviations from the mean: (2-6)² = 16, (4-6)² = 4, (6-6)² = 0, (8-6)² = 4, (10-6)² = 16
    • Variance = (16 + 4 + 0 + 4 + 16) / 5 = 40 / 5 = 8

Standard Deviation

  • The **standard deviation** is the square root of the variance. It provides a measure of the spread of data points in the same unit as the data.
  • Formula for **Population Standard Deviation**: Standard   Deviation ( σ ) = Σ(Xi - μ)2 N
  • Formula for **Sample Standard Deviation**: Standard   Deviation ( s ) = Σ(Xi - X̄)2 n - 1
  • Steps to Calculate Standard Deviation:
    • Find the variance of the dataset.
    • Take the square root of the variance.
  • Example (Population Standard Deviation):
    • Given the dataset: 2, 4, 6, 8, 10, and variance = 8, the standard deviation is:
    • Standard Deviation = √8 ≈ 2.83

Comparison of Variance and Standard Deviation

Measure Variance Standard Deviation
Definition Average of squared deviations from the mean. Square root of the variance.
Units Squared units of the data (e.g., square meters). Same units as the original data (e.g., meters).
Interpretation Harder to interpret due to squared units. Easier to interpret as it is in the same units as the data.
Sensitivity to Outliers Sensitive to outliers. Sensitive to outliers.
Usage Used in statistical calculations (e.g., ANOVA, regression). Used in real-world interpretation and comparisons.

Relationship Between Variance and Standard Deviation

  • The **standard deviation** is the square root of the **variance**.
  • This relationship is important because variance is in squared units, making it harder to interpret, while the standard deviation is in the same units as the data, making it easier to understand.

4. Permutations and Combinations

Definition

  • Permutations and Combinations are fundamental concepts in combinatorics, the branch of mathematics dealing with counting and arrangements.
  • These concepts are used to determine the number of ways to arrange or select items from a set.

Permutations

  • A **permutation** is an arrangement of objects in a specific order. The order matters in permutations.
  • Formula for **Permutations**: P ( n , r ) = n ! (n - r) !

    Where:
    n = Total number of objects,
    r = Number of objects to arrange,
    ! (Factorial) is the product of all positive integers up to that number (e.g., 5! = 5 × 4 × 3 × 2 × 1 = 120).

  • Example:
    • If you have 5 books and want to know how many ways you can arrange 3 of them on a shelf:
    • P ( 5 , 3 ) = 5 ! (5 - 3) !

      Which simplifies to: 5 × 4 × 3 = 60 ways to arrange 3 books from 5.

  • Key Points About Permutations:
    • The order of arrangement matters.
    • Used for arranging items in a specific order.
    • If the objects are identical, divide by the factorial of the number of identical objects.

Combinations

  • A **combination** is a selection of objects where the order does not matter.
  • Formula for **Combinations**: C ( n , r ) = n ! r ! (n - r) !

    Where:
    n = Total number of objects,
    r = Number of objects to choose,
    ! (Factorial) is the product of all positive integers up to that number.

  • Example:
    • If you have 5 books and want to know how many ways you can choose 3 books from them (without considering order):
    • C ( 5 , 3 ) = 5 ! 3 ! 2 !

      Which simplifies to: 5 × 4 / 2 × 1 = 10 ways to choose 3 books from 5.

  • Key Points About Combinations:
    • The order of selection does not matter.
    • Used for selecting items without regard to order.
    • The formula accounts for the fact that the same selection can appear in different orders.

Key Differences Between Permutations and Combinations

Aspect Permutations Combinations
Order Matters Yes No
Formula P ( n , r ) = n ! (n - r) ! C ( n , r ) = n ! r ! (n - r) !
Used For Arranging objects in a specific order. Selecting objects without regard to order.
Example How many ways can 3 books be arranged from 5? How many ways can 3 books be chosen from 5?

Factorial (n!)

  • The **factorial** of n (denoted as n!) is the product of all positive integers from 1 to n. It is used in both permutations and combinations formulas.
  • Examples:
    • 5! = 5 × 4 × 3 × 2 × 1 = 120
    • 3! = 3 × 2 × 1 = 6
    • 0! = 1 (By definition, the factorial of 0 is 1)

Special Cases

  • Permutations of Identical Objects: If there are identical objects in the set, divide by the factorial of the number of identical objects. For example, the number of permutations of "AAB" is:
    • P = 3! / 2! = 6 / 2 = 3
  • Combinations with Repetition: When repetition of objects is allowed, the number of combinations is given by:
    • C ′ ( n , r ) = (n + r - 1) ! r ! (n - 1) !
  • Permutations with Repetition: When repetition is allowed, the number of permutations is:
    • P′(n, r) = nr
    • Example: How many ways can you arrange 3 digits from {1, 2, 3} with repetition? P′(3, 3) = 33 = 27

5. Probability Distributions

Definition

  • A probability distribution describes how the probabilities are distributed over the values of a random variable.
  • It provides a function that gives the probability of each possible outcome in an experiment or random process.

Types of Probability Distributions

  1. Discrete Probability Distribution

    • The random variable can only take distinct, separate values.
    • The sum of all probabilities in a discrete distribution is always equal to 1.
    • Example: Consider the roll of a fair six-sided die. The random variable X represents the outcome of the die roll, which can take values 1, 2, 3, 4, 5, 6, each with a probability of 1/6.
    • Formula: P X = 1 6

  2. Continuous Probability Distribution

    • The random variable can take any value within a given range or interval.
    • Example: A normal distribution for heights, where the variable can take any value within a continuous range.
    • The probability of any specific value is 0, and probabilities are calculated for a range of values.
    • Formula to find probability within range: P ( a ≤ X ≤ b ) = ∫ a b f ( x ) dx

Common Types of Probability Distributions

  1. Binomial Distribution (Discrete)

    • Models the number of successes in a fixed number of independent Bernoulli trials.
    • Formula: P ( X = k ) = n k p k (1-p) (n-k)

  2. Poisson Distribution (Discrete)

    • Models the number of events occurring within a fixed interval of time or space.
    • Formula: P ( X = k ) = λ k k ! e -λ

  3. Normal Distribution (Continuous)

    • The most common continuous distribution, characterized by the mean μ and standard deviation σ.
    • Formula: f ( x ) = 1 σ 2 π e - (x - μ) 2 2 σ 2

  4. Uniform Distribution (Continuous)

    • All outcomes are equally likely within a given range [a, b].
    • Formula: f ( x ) = 1 b - a

  5. Exponential Distribution (Continuous)

    • Models the time between events in a Poisson process.
    • Formula: f ( x ) = λ e - λx
