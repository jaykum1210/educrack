Chapter 2 - Complex Numbers & Quadratic Equations

1. Complex Numbers, Argand Plane, Modulus & Argument

i. Complex Numbers

A complex number is a number of the form:

z = a + bi, where:

  • a is the real part (Re(z))
  • b is the imaginary part (Im(z))
  • i (called iota) is the imaginary unit, defined as i² = -1

Operations on Complex Numbers:

  • Addition: (a + bi) + (c + di) = (a + c) + (b + d)i
  • Subtraction: (a + bi) - (c + di) = (a - c) + (b - d)i
  • Multiplication: (a + bi) × (c + di) = ac - bd + (ad + bc)i (since i² = -1)
  • Division: (a + bi) / (c + di) = (a + bi)(c - di) / (c² + d²)

ii. Argand Plane (Complex Plane)

The Argand plane is a way to represent complex numbers graphically.

  • The x-axis represents the real part.
  • The y-axis represents the imaginary part.
  • A complex number z = a + bi is plotted as the point (a, b) in the plane.

Examples:

  • z = 3 + 4i is plotted at (3, 4)
  • z = -2 - i is plotted at (-2, -1)

iii. Modulus of a Complex Number

The modulus (absolute value) of a complex number z = a + bi is given by:

|z| = √(a² + b²)

It represents the distance from the origin in the Argand plane.

Example:

For z = 3 + 4i, the modulus is:

|z| = √(3² + 4²) = √(9 + 16) = √25 = 5

iv. Argument of a Complex Number

The argument (or phase) of a complex number is the angle θ it makes with the positive x-axis, given by:

θ = tan⁻¹(b / a)

  • Principal argument: -π < θ ≤ π
  • General argument: θ + 2kπ, k ∈ Z

Example:

For z = 1 + i, the argument is:

θ = tan⁻¹(1 / 1) = tan⁻¹(1) = π/4

v. Polar Form of a Complex Number

A complex number can also be written in polar form:

z = r (cos θ + i sin θ), where:

  • r = |z| is the modulus
  • θ is the argument

Example:

For z = 1 + i:

  • r = √(1² + 1²) = √2
  • θ = π/4

Thus, z = √2 (cos π/4 + i sin π/4)

2. Quadratic Equations, Roots & Coefficients

i. Quadratic Equation

A quadratic equation is a second-degree polynomial equation in a single variable, generally written in the form:

ax² + bx + c = 0, where:

  • a, b, and c are constants (real or complex numbers),
  • x is the variable,
  • a ≠ 0 (because if a = 0, it becomes a linear equation).

ii. Roots of a Quadratic Equation

The roots of a quadratic equation are the values of x that satisfy the equation ax² + bx + c = 0. These can be found using various methods:

  • Factoring (when possible),
  • Completing the square,
  • Quadratic Formula: This is the most widely used method to find the roots, and it is derived from completing the square. The quadratic formula is given by:

x = (-b ± √(b² - 4ac)) / 2a

Where:

  • b² - 4ac is called the discriminant.

iii. Discriminant

The discriminant (Δ) is the expression b² - 4ac under the square root in the quadratic formula. It provides information about the nature of the roots of the quadratic equation:

  • If Δ > 0: The equation has two real and distinct roots.
  • If Δ = 0: The equation has two real and identical roots (or one real root).
  • If Δ < 0: The equation has no real roots, but two complex roots.

iv. Relationship Between Roots and Coefficients (Vieta's Formula)

For a quadratic equation ax² + bx + c = 0, the following relationships exist between the roots and coefficients:

  • Let the roots be r₁ and r₂. From Vieta's formulas:
  • Sum of the roots: r₁ + r₂ = -b / a
  • Product of the roots: r₁ ⋅ r₂ = c / a

v. Coefficients

In a quadratic equation ax² + bx + c = 0:

  • a is the coefficient of x², determining the parabola's "width" and "direction."
  • b is the coefficient of x, affecting the slope and position of the parabola.
  • c is the constant term, determining the vertical position of the parabola.

vi. Properties of Quadratic Equations

The graph of a quadratic equation is a parabola. The following are key properties:

  • If a > 0, the parabola opens upward, and if a < 0, it opens downward.
  • The vertex of the parabola (the point where it reaches its maximum or minimum) occurs at:

x = -b / 2a

At this point, the value of y (or the value of the quadratic equation) is given by:

y = (4ac - b²) / 4a

vii. Example

Solve the quadratic equation 2x² - 4x - 6 = 0.

Identify coefficients:

  • a = 2, b = -4, c = -6

Find the discriminant:

Δ = b² - 4ac = (-4)² - 4(2)(-6) = 16 + 48 = 64

Find the roots using the quadratic formula:

x = (-(-4) ± √64) / (2(2)) = (4 ± 8) / 4

  • x = (4 + 8) / 4 = 12 / 4 = 3
  • x = (4 - 8) / 4 = -4 / 4 = -1

Thus, the roots are x = 3 and x = -1.
