Chapter 7 - Limits, Continuity & Differentiability

1. Limits

Definition

  • The limit of a function \( f(x) \) as \( x \) approaches \( a \) is the value that \( f(x) \) approaches as \( x \) gets closer to \( a \). Denoted as:

    • lim x → a f ( x ) = L

Types of Limits

  • Finite Limit: When \( f(x) \) approaches a finite value \( L \). Example:

    • lim x → 2 (3x + 1) = 7

  • Infinite Limit: When \( f(x) \) grows without bound as \( x \) approaches \( a \). Example:

    • lim x → 0 1 x = ∞

  • One-Sided Limit: Limit from one direction:
    • Left-hand limit: lim x → a âˆ’ f ( x )
    • Right-hand limit: lim x → a + f ( x )
  • Limit at Infinity: As \( x \to \infty \) or \( x \to -\infty \), the function approaches a value. Example:

    • lim x → ∞ 1 x = 0

Properties of Limits

  • Sum Rule: lim x → a (f(x) + g(x)) = lim x → a f(x) + lim x → a g(x)
  • Product Rule: lim x → a (f(x) â‹… g(x)) = lim x → a f(x) â‹… lim x → a g(x)
  • Quotient Rule: lim x → a f(x) g(x) = lim x → a f(x) lim x → a g(x)
  • Constant Multiple Rule: lim x → a (c â‹… f(x)) = c â‹… lim x → a f(x)
  • Power Rule: lim x → a [f(x)] n = lim x → a f(x) n

Common Techniques

  • Direct Substitution: Substitute \( x = a \) directly into the function.

    • Example: lim x → 2 (3x + 1) = 7

  • Factoring: Factor functions to simplify and eliminate indeterminate forms.

  • L'Hopital's Rule: For indeterminate forms \( \frac{0}{0} \) or \( \frac{\infty}{\infty} \), differentiate the numerator and denominator.
  • Squeeze Theorem: If \( f(x) \leq g(x) \leq h(x) \) and the limits of \( f(x) \) and \( h(x) \) are the same, then the limit of \( g(x) \) is also the same.

2. Continuity

Definition of Continuity

  • A function \( f(x) \) is continuous at \( x = a \) if:
    • Defined at \( a \): \( f(a) \) exists.
    • Limit exists at \( a \): lim x → a f(x) exists.
    • Function value equals the limit: lim x → a f(x) = f(a) .

Types of Discontinuities

  • Removable: The limit exists but the function is not defined or not equal to the limit at \( x = a \).

    • Example: f(x) = x Â² âˆ’ 1 x âˆ’ 1 at x = 1

  • Jump: Left-hand and right-hand limits are not equal at \( x = a \).

    • Example: Piecewise functions with different values on each side.

  • Infinite: Function approaches infinity as \( x \) approaches \( a \).

    • Example: f(x) = 1 x at x = 0

Theorems

  • Intermediate Value Theorem: If a function is continuous on \( [a, b] \), it takes every value between \( f(a) \) and \( f(b) \).
  • Sum, Product, and Quotient Theorems: If \( f(x) \) and \( g(x) \) are continuous at \( a \), so are:
    • \( f(x) + g(x) \),
    • \( f(x) \cdot g(x) \),
    • \( \frac{f(x)}{g(x)} \) (if \( g(a) \neq 0 \)).

Key Continuous Functions

  • Polynomials: Continuous everywhere.

    • Example: f(x) = x 2

  • Rational Functions: Continuous where the denominator is non-zero.

    • Example: f(x) = 1 x (continuous for x â‰  0)

  • Trigonometric Functions: Continuous where defined.

    • Example: f(x) = sin(x)

3. Differentiability in Calculus

Definition of Differentiability

  • A function \( f(x) \) is differentiable at \( x = a \) if the derivative exists at that point. Mathematically, the derivative of \( f(x) \) at \( x = a \) is:
    • f ' ( a ) = lim h → 0 f(a+h) âˆ’ f(a) h

Differentiability vs Continuity

  • A function must be continuous at \( x = a \) to be differentiable there, but a continuous function may not be differentiable.
  • Differentiability implies continuity, but the reverse is not always true. For example, a function can be continuous at a point but not differentiable there (like \( f(x) = |x| \) at \( x = 0 \)).

Conditions for Differentiability

  • Continuity at the point: The function must be continuous at \( x = a \).
  • No sharp corners: The function must not have any sharp corners or cusps at \( x = a \).
  • No vertical tangents: The function must not have a vertical tangent at \( x = a \).

Differentiability of Common Functions

  • Polynomials: All polynomial functions are differentiable everywhere.

    • Example: f(x) = x 3 âˆ’ 3 x + 2

  • Rational Functions: Differentiable where the denominator is non-zero.

    • Example: f(x) = 1 x for x â‰  0

  • Trigonometric Functions: Basic trigonometric functions like \( \sin(x) \), \( \cos(x) \), and \( \tan(x) \) are differentiable wherever they are defined.

    • Example: f(x) = sin(x)

Examples of Differentiability

  • Example 1: \( f(x) = x^2 \) is differentiable everywhere, and its derivative is \( f'(x) = 2x \).
  • Example 2: \( f(x) = |x| \) is not differentiable at \( x = 0 \) due to a sharp corner.
  • Example 3: \( f(x) = \frac{1}{x} \) is differentiable for \( x â‰  0 \), but not at \( x = 0 \).

Applications of Differentiation

  • Rate of Change: Differentiation is used to find the rate at which a quantity changes with respect to another variable. For example, if the position of an object is given by \( s(t) \), the derivative \( \frac{ds}{dt} \) gives the velocity, which is the rate of change of position with respect to time.

    Example: If the position function is \( s(t) = t^2 + 3t \), the velocity is: v ( t ) = d s / d t = 2 t + 3

  • Maxima and Minima (Optimization): Differentiation is used to find the critical points of a function where the function reaches local maximum or minimum values. These points are found by setting the first derivative equal to zero (\( f'(x) = 0 \)) and analyzing the second derivative.
    • Steps to Find Maxima/Minima:
      1. Find the first derivative: \( f'(x) \)
      2. Set the first derivative equal to zero: Solve \( f'(x) = 0 \) to find critical points.
      3. Find the second derivative: \( f''(x) \)
      4. Determine the nature of the critical points:
        • If \( f''(x) > 0 \), the function has a local minimum.
        • If \( f''(x) < 0 \), the function has a local maximum.
        • If \( f''(x) = 0 \), further analysis is needed.
    • Example: For \( f(x) = x^3 - 3x^2 + 2 \):
      • First derivative: \( f'(x) = 3x^2 - 6x \)
      • Set \( f'(x) = 0 \): \( 3x(x - 2) = 0 \Rightarrow x = 0 \text{ or } x = 2 \)
      • Second derivative: \( f''(x) = 6x - 6 \)
      • At \( x = 0 \), \( f''(0) = -6 \), indicating a local maximum.
      • At \( x = 2 \), \( f''(2) = 6 \), indicating a local minimum.
