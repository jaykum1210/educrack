Chapter 15 - Trigonometry

1. Trigonometric Identities and Equations - Short Notes

Trigonometric Identities

Basic Identities:

  • Reciprocal Identities:
    • sin(θ) = 1/csc(θ)
    • cos(θ) = 1/sec(θ)
    • tan(θ) = 1/cot(θ)

Pythagorean Identities:

  • sin²(θ) + cos²(θ) = 1
  • 1 + tan²(θ) = sec²(θ)
  • 1 + cot²(θ) = csc²(θ)

Quotient Identities:

  • tan(θ) = sin(θ) / cos(θ)
  • cot(θ) = cos(θ) / sin(θ)

Sum and Difference Identities:

Sine:

  • sin(A ± B) = sin(A)cos(B) ± cos(A)sin(B)

Cosine:

  • cos(A ± B) = cos(A)cos(B) ∓ sin(A)sin(B)

Tangent:

  • tan(A ± B) = (tan(A) ± tan(B)) / (1 ∓ tan(A)tan(B))

Double Angle Identities:

Sine:

  • sin(2A) = 2sin(A)cos(A)

Cosine:

  • cos(2A) = cos²(A) − sin²(A)

Tangent:

  • tan(2A) = (2tan(A)) / (1 − tan²(A))

Trigonometric Equations

Basic Equations:

  • Example 1:

    sin(x) = 1/2 ⇒ x = π/6, 5π/6

  • Example 2:

    cos(x) = −1/2 ⇒ x = 2π/3, 4π/3

Equations with Multiple Functions:

  • Example 1:

    tan(x) = 1 ⇒ x = π/4, 5π/4

  • Example 2:

    sin²(x) = 1/2 ⇒ x = π/4, 3π/4, 5π/4, 7π/4

Using Identities:

  • Example:

    sin²(x) − cos²(x) = 0 ⇒ cos(2x) = 0 ⇒ x = π/4 + nπ/2

2. Trigonometric Functions, Inverse Trigonometric Functions, and Their Properties - Short Notes

i. Trigonometric Functions:

Trigonometric functions are functions of angles and are fundamental in the study of geometry, particularly in right triangles and circular motion.

Sine:

sin(θ) = opposite / hypotenuse

Cosine:

cos(θ) = adjacent / hypotenuse

Tangent:

tan(θ) = opposite / adjacent = sin(θ) / cos(θ)

Cosecant:

csc(θ) = 1 / sin(θ)

Secant:

sec(θ) = 1 / cos(θ)

Cotangent:

cot(θ) = 1 / tan(θ) = cos(θ) / sin(θ)

ii. Inverse Trigonometric Functions:

Inverse trigonometric functions are the inverse operations of the standard trigonometric functions. They give the angle corresponding to a given trigonometric value.

Inverse Sine (sin−1 x or arcsin x):

sin−1 x = θ if sin(θ) = x, −π/2 ≤ θ ≤ π/2

Inverse Cosine (cos−1 x or arccos x):

cos−1 x = θ if cos(θ) = x, 0 ≤ θ ≤ π

Inverse Tangent (tan−1 x or arctan x):

tan−1 x = θ if tan(θ) = x, −π/2 < θ < π/2

Inverse Cosecant (csc−1 x):

csc−1 x = θ if csc(θ) = x, −π/2 ≤ θ ≤ π/2, θ ≠ 0

Inverse Secant (sec−1 x):

sec−1 x = θ if sec(θ) = x, 0 ≤ θ ≤ π, θ ≠ π/2

Inverse Cotangent (cot−1 x):

cot−1 x = θ if cot(θ) = x, 0 < θ < π

iii. Properties of Trigonometric Functions:

Periodicity:

  • Sine and Cosine: Period 2π
  • Tangent and Cotangent: Period π
  • Secant and Cosecant: Period 2π

Range:

  • Sine, Cosine: [−1, 1]
  • Tangent, Cotangent: (−∞, ∞)
  • Secant, Cosecant: (−∞, −1] ∪ [1, ∞)

Symmetry:

  • Sine and Tangent: Odd functions, sin(−θ) = −sin(θ), tan(−θ) = −tan(θ)
  • Cosine, Secant, Cotangent: Even functions, cos(−θ) = cos(θ), sec(−θ) = sec(θ)

iv. Properties of Inverse Trigonometric Functions:

Range:

  • sin−1 x: [−π/2, π/2]
  • cos−1 x: [0, π]
  • tan−1 x: (−π/2, π/2)
  • csc−1 x: [−π/2, π/2] (excluding 0)
  • sec−1 x: [0, π] (excluding π/2)
  • cot−1 x: (0, π)

Identity Relations:

  • sin(sin−1 x) = x, for x ∈ [−1, 1]
  • cos(cos−1 x) = x, for x ∈ [−1, 1]
  • tan(tan−1 x) = x, for x ∈ (−∞, ∞)
  • csc(csc−1 x) = x, for |x| ≥ 1
  • sec(sec−1 x) = x, for |x| ≥ 1
  • cot(cot−1 x) = x, for x ∈ (−∞, ∞)

v. Inverse Trigonometric Function Equations:

To solve equations involving inverse trigonometric functions, it is useful to express the inverse function in terms of the original trigonometric function.

Example:

Solve sin−1 x = π/6:

This implies sin(π/6) = x, thus x = 1/2

3. Heights and Distances

Definition

  • Heights and Distances is a branch of Trigonometry used to calculate the height, distance, or angle of elevation/depression of objects using trigonometric principles.

Key Concepts

  • Angle of Elevation: The angle formed by the line of sight from the observer to the top of an object above the observer.
  • Angle of Depression: The angle formed by the line of sight from the observer to an object below the observer.

Basic Trigonometric Ratios

  • For a right triangle with angle θ, base b, height h, and hypotenuse l, the following relations are useful:
    • sin ⁡ θ = opposite hypotenuse
    • cos ⁡ θ = adjacent hypotenuse
    • tan ⁡ θ = opposite adjacent

Important Formulas

  • Height of an Object (using tanθ): When the distance from the observer to the object and the angle of elevation are known: h = d ⋅ tan ⁡ θ
  • Distance to an Object (using tanθ): When the height of the object and the angle of elevation are known: d = h tan ⁡ θ
  • Height of an Object (using sinθ or cosθ): If the distance to the object (or the length of the base) and the angle of elevation are known: h = l ⋅ sin ⁡ θ

Solving Problems in Heights and Distances

  • Example 1: A person standing at a distance of 30 m from a tower observes that the angle of elevation to the top of the tower is 30°. Find the height of the tower.
    • Using the formula: h = d ⋅ tan ⁡ 30°
    • Answer: The height of the tower is approximately 17.31 meters.
  • Example 2: A man is standing on the ground, looking up at the top of a building. The angle of elevation is 45°, and the height of the building is 50 meters. Find the distance between the man and the building.
    • Using the formula: d = h tan ⁡ 45°
    • Answer: The distance from the man to the building is 50 meters.

Applications of Heights and Distances

  • Navigation: Used to determine the distance between two points using angles of elevation or depression.
  • Architecture: Determining the height of buildings, towers, and other structures using measurements at ground level.
  • Surveying: Used to determine the height of mountains, bridges, and other large structures.
