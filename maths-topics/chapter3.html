Chapter 3 - Matrices & Determinants

1. Types & Operations on Matrices

Definition

  • A matrix is a rectangular array of numbers arranged in rows and columns.
  • It is typically denoted by a capital letter, and its elements are denoted by lowercase letters with two subscripts: a ij , where i represents the row and j represents the column.

Types of Matrices

  1. Row Matrix

    • A matrix with only one row, i.e., it is of the form 1 ×n .
    • Example: a 11 a 12 a 13 a 1n

  2. Column Matrix

    • A matrix with only one column, i.e., it is of the form m ×1 .
    • Example: a 11 a 21 a 31 ⋮ a m1

  3. Square Matrix

    • A matrix where the number of rows equals the number of columns (e.g., n ×n ).
    • Example: a 11 a 12 ⋮ a n1

  4. Zero (Null) Matrix

    • A matrix in which all elements are zero. It can be of any size.
    • Example: 0 0 0 0

  5. Identity Matrix

    • A square matrix where all the diagonal elements are 1, and all other elements are 0.
    • Example: 1 0 0 1 0 0

Operations on Matrices

Matrix Addition

  • Two matrices can be added if and only if they have the same dimensions.
  • Formula: C ij = A ij + B ij

Matrix Subtraction

  • Matrix subtraction is similar to addition, only possible when matrices have the same dimensions.
  • Formula: C ij = A ij - B ij

Scalar Multiplication

  • A matrix can be multiplied by a scalar.
  • Formula: B ij = k ⋅ A ij

Matrix Multiplication

  • Matrix multiplication is only possible if the number of columns of the first matrix equals the number of rows of the second matrix.
  • Formula: C ij = k = 1 ⋯ n A ik B kj

2. Determinants: Properties & Area of a Triangle

i. Determinants

  • A determinant is a scalar value that is computed from the elements of a square matrix. It provides useful properties about the matrix, such as whether the matrix is invertible or singular (non-invertible).
  • For a square matrix, the determinant is often denoted as det ( A ) or | A | .

ii. Determinant of a 2x2 Matrix

For a 2x2 matrix:

A 2×2 = a 11 a 12 a 21 a 22

The determinant of A is calculated as:

det ( A ) = a ₁₁ a ₂₂ - a ₁₂ a ₂₁

iii. Determinant of a 3x3 Matrix

For a 3x3 matrix:

A 3×3 = a 11 a 12 a 13 a 21 a 22 a 23 a 31 a 32 a 33

The determinant of A is calculated as:

det ( A ) = a ( e i - f h ) - b ( d i - f g ) + c ( d h - e g )

iv. Properties of Determinants

  • Determinant of a Diagonal or Scalar Matrix: The determinant of a diagonal matrix (or scalar matrix) is the product of the diagonal elements. For example, for a diagonal matrix: A 2×2 = a 11 a 22 ⇒ det ( A ) = a ⋅ b
  • Determinant of the Identity Matrix: The determinant of the identity matrix is 1. det ( I ) = 1
  • Determinant of a Zero Matrix: The determinant of a zero matrix is always 0. det ( 0 ) = 0
  • Row Interchanges: If two rows (or two columns) of a matrix are interchanged, the sign of the determinant changes (it gets multiplied by -1).
  • Determinant of a Matrix Product: The determinant of the product of two matrices is the product of their determinants: det ( A B ) = det ( A ) ⋅ det ( B )
  • Scalar Multiplication: If each element of a row (or column) of a matrix is multiplied by a scalar k , the determinant of the matrix is multiplied by k raised to the power of n for an n × n matrix: det ( kA ) = k ⁿ ⋅ det ( A )
  • Singular Matrix: A matrix is singular (non-invertible) if and only if its determinant is 0. That is, det ( A ) = 0 .

3. Inverse of a Matrix & Solving Linear Equations

Definition

  • The inverse of a matrix \( A \) is a matrix \( A^{-1} \) such that when multiplied with \( A \), it results in the identity matrix \( I \).
  • Formula: \( A \times A^{-1} = A^{-1} \times A = I \)

Conditions for Invertibility

  • The matrix \( A \) has an inverse if and only if \( \det(A) \neq 0 \).
  • If \( \det(A) = 0 \), the matrix is called singular and does not have an inverse.

Inverse of a 2x2 Matrix

  • For a 2x2 matrix:
    • A = a b c d
  • The inverse of matrix \( A \), denoted as \( A^{-1} \), is given by:
    • A -1 = 1 det(A) d -b -c a

Example of Inverse of a 2x2 Matrix

  • Given the matrix:
    • A = 2 3 1 4
  • The determinant of \( A \) is:
    • det(A) = (2)(4) - (3)(1) = 5
  • Since \( \det(A) = 5 \neq 0 \), the inverse exists:
    • A -1 = 1 5 4 -3 -1 2

Solving Linear Equations Using the Inverse Matrix

  • Consider the system of linear equations:
    • a 1 x + b 1 y = c 1 a 2 x + b 2 y = c 2
  • This can be written as a matrix equation:
    • A · X = C
  • To solve for \( X \), we multiply both sides by \( A^{-1} \):
    • A -1 · A · X = A -1 · C
  • Since \( A^{-1} \cdot A = I \), we get:
    • X = A -1 · C

Example: Solving a System of Equations

  • Given the system:
    • 2x + 3y = 5 x + 4y = 6
  • In matrix form:
    • A = 2 3 1 4
  • Multiplying the inverse of \( A \) by \( C \) gives:
    • X = A -1 · C

General Case for Solving Linear Equations

  • For a system of \( n \) linear equations in \( n \) variables:
    • A · X = C
  • The solution is given by:
    • X = A -1 · C
