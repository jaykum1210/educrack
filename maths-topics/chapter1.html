Chapter 1 - Sets, Relations & Functions

1. Sets, Operations, and Power Set

i. Sets

A set is a collection of distinct objects, called elements, written inside {}.

Types of Sets:

  • Empty Set (∅): A set with no elements.
  • Finite & Infinite Sets: A set with a countable or uncountable number of elements.
  • Subset (⊆): A set A is a subset of B if all elements of A are in B.
  • Proper Subset (⊂): A subset that is not equal to the given set.
  • Power Set: The set of all subsets of a given set.
  • Universal Set (U): A set containing all elements under discussion.

ii. Operations on Sets

  • Union (∪): A set containing all elements from A and B.
    Example: If A = {1,2,3}, B = {3,4,5}, then A ∪ B = {1,2,3,4,5}
  • Intersection (∩): A set containing common elements of A and B.
    Example: A ∩ B = {3}
  • Difference (-): A set containing elements in A but not in B.
    Example: A - B = {1,2}
  • Complement (A’): Elements in the universal set but not in A.
  • Symmetric Difference (Δ): Elements in either A or B but not both.
    Example: A Δ B = {1,2,4,5}

iii. Power Set

Definition: The set of all subsets of a given set.

Formula: If A has n elements, then P(A) has 2^n subsets.

Example: If A = {1,2}, then P(A) = {∅, {1}, {2}, {1,2}}.

Properties:

  • Always includes {} and the set itself.
  • If |A| = n, then |P(A)| = 2^n.

2. Relations

A relation between two sets A and B is a subset of their Cartesian product A × B. It defines a relationship between elements of A and B.

Types of Relations

  • Empty Relation: No element of A is related to any element of B.
  • Universal Relation: Every element of A is related to every element of B.
  • Reflexive Relation: If (a, a) ∈ R for every a ∈ A.
  • Symmetric Relation: If (a, b) ∈ R implies (b, a) ∈ R.
  • Transitive Relation: If (a, b) ∈ R and (b, c) ∈ R, then (a, c) ∈ R.
  • Equivalence Relation: A relation that is reflexive, symmetric, and transitive.

Equivalence Relation

A relation R on a set A is an equivalence relation if it satisfies:

  • Reflexivity: (a, a) ∈ R for all a ∈ A
  • Symmetry: If (a, b) ∈ R, then (b, a) ∈ R
  • Transitivity: If (a, b) ∈ R and (b, c) ∈ R, then (a, c) ∈ R

Example:

Consider a relation R on set A = {1,2,3,4} defined by a R b if a mod 2 = b mod 2.

  • Reflexive: Every element is related to itself.
  • Symmetric: If a R b, then b R a.
  • Transitive: If a R b and b R c, then a R c.

Thus, R is an equivalence relation.

3. Functions: One-One, Onto, and Composition

A function is a special type of relation where each element of a set A (domain) is assigned exactly one element of a set B (codomain). It is denoted as:

f: A → B, where A is the domain, B is the codomain, and f(x) is the output for each x in A.

i. One-One (Injective) Function

A function is called one-one (injective) if different elements in A map to different elements in B. This means no two different inputs have the same output.

Mathematical Definition: If f(a1) = f(a2), then a1 = a2 (If outputs are the same, then inputs must also be the same).

Example:

Let f(x) = 2x + 3. If f(a1) = f(a2), then 2a1 + 3 = 2a2 + 3, which simplifies to a1 = a2. Since different values of x give different outputs, it is one-one.

Graphical Representation:

A function is one-one if any horizontal line cuts the graph at most once.

  • f(x) = x and f(x) = e^x are one-one.
  • f(x) = x² is not one-one because f(-x) = f(x).

ii. Onto (Surjective) Function

A function is called onto (surjective) if every element in the codomain B has at least one preimage in domain A. This means that the function covers all elements of B.

Mathematical Definition: For every b ∈ B, there exists an a ∈ A such that f(a) = b.

Example:

Let f(x) = x³ from ℝ → ℝ. Since every real number has a cube root, every y ∈ ℝ has a preimage x = y³. Thus, the function is onto.

Graphical Representation:

A function is onto if its graph covers the entire range of the codomain.

  • f(x) = x² from ℝ → ℝ is not onto since negative numbers are missing from the range.
  • f(x) = x³ is onto, as it covers all real numbers.

iii. One-One and Onto (Bijective) Function

A function is called bijective if it is both one-one and onto.

Properties of a Bijective Function:

  • It has a unique output for every input (one-one).
  • It covers the entire codomain (onto).
  • It has an inverse function f⁻¹ such that f⁻¹(f(x)) = x.

Example:

f(x) = x + 2 from ℝ → ℝ is bijective, since it is both injective and surjective.

f(x) = x² is not bijective because it is neither one-one nor onto.

iv. Composition of Functions

The composition of functions is applying one function after another. If we have two functions:

  • f: A → B
  • g: B → C

Then the composition is written as:

(g ∘ f)(x) = g(f(x)), meaning that we first apply f(x), then apply g(x) to the result.

Example of Function Composition:

Let:

  • f(x) = x + 2
  • g(x) = x²

Then, (g ∘ f)(x) = g(f(x)) = (x + 2)²

Properties of Function Composition:

  • Associativity: (h ∘ g) ∘ f = h ∘ (g ∘ f) — Applying functions in different orders gives the same result.
  • Identity Function: I(x) = x, if I(x) is the identity function, then f ∘ I = I ∘ f = f.
  • Inverse Function: If f: A → B and g: B → A such that g(f(x)) = x, then g is the inverse of f.

Example of Inverse Function:

If f(x) = 2x + 3, then solving for x gives x = (y - 3)/2, and thus f⁻¹(x) = (x - 3)/2.
