1. Vector and Scalar

Definition

  • Scalars: A scalar is a quantity that is fully described by a single numerical value and its unit. Scalars only have magnitude (size) and no direction.
  • Vectors: A vector is a quantity that is described by both magnitude and direction. It is represented by an arrow, where the length of the arrow represents the magnitude, and the direction of the arrow represents the direction of the vector.

Examples

Scalars:

  • Mass (kg)
  • Temperature (¬∞C)
  • Speed (m/s)
  • Time (s)
  • Distance (m)

Vectors:

  • Velocity (m/s)

2. Components of Vectors

Definition

  • A vector can be broken down into components along specific directions, usually along the coordinate axes of a system (typically the x-axis, y-axis, and z-axis in 3D space).
  • The process of breaking a vector into components simplifies vector operations and allows for easier analysis in different directions.

Components of a Vector in 2D

  • In a two-dimensional Cartesian coordinate system, any vector A can be written as a sum of its components along the x-axis and y-axis.
  • Consider a vector A having magnitude A and direction Œ∏ (angle with the positive x-axis).
  • The vector A can be represented as:

    A = Ax i + Ay j

    • Ax is the x-component of the vector A
    • Ay is the y-component of the vector A
    • i and j are unit vectors in the direction of the x-axis and y-axis, respectively.
  • Using trigonometry, the components of vector A are given by:
    • Ax = A * cos(Œ∏) (the x-component)
    • Ay = A * sin(Œ∏) (the y-component)
  • Thus, the vector A in component form becomes:

    A = Ax i + Ay j = A * cos(Œ∏) i + A * sin(Œ∏) j

Example (2D)

  • If A has a magnitude of 5 units and makes an angle of 30¬∞ with the positive x-axis, then:
    • Ax = 5 * cos(30¬∞) = 5 * ‚àö3/2 = 4.33
    • Ay = 5 * sin(30¬∞) = 5 * 1/2 = 2.5
  • So, the vector A in component form is:

    A = 4.33 i + 2.5 j

Components of a Vector in 3D

  • In a three-dimensional Cartesian coordinate system, any vector A can be written as a sum of its components along the x-axis, y-axis, and z-axis.
  • Consider a vector A in 3D space, which can be represented as:

    A = Ax i + Ay j + Az k

    • Ax is the x-component of the vector A
    • Ay is the y-component of the vector A
    • Az is the z-component of the vector A
    • i, j, k are unit vectors along the x-axis, y-axis, and z-axis, respectively.
  • If you know the magnitude A of the vector and the angles Œ∏x, Œ∏y, and Œ∏z that the vector makes with the coordinate axes, you can find the components as follows:
    • Ax = A * cos(Œ∏x)
    • Ay = A * cos(Œ∏y)
    • Az = A * cos(Œ∏z)
  • Alternatively, if the direction cosines are given, the components can be directly calculated using:
    • Ax = A * cos(Œ±)
    • Ay = A * cos(Œ≤)
    • Az = A * cos(Œ≥)

    • Az = 6 * cos(90¬∞) = 6 * 0 = 0
  • Thus, the vector A in component form is:

    A = 4.24 i + 3 j + 0 k

Unit Vectors

  • Unit vectors are vectors with a magnitude of 1 and are used to define the direction of a vector in space.
  • In the Cartesian coordinate system, the unit vectors are:
    • i along the x-axis
    • j along the y-axis
    • k along the z-axis

Rectangular Components of a Vector

  • The components of a vector can be considered as projections of the vector along the coordinate axes. This is useful for vector operations such as addition, subtraction, and scalar multiplication.

Component Form of a Vector (General Case)

  • In general, a vector A can be written in terms of its components along the axes as:

    A = Ax i + Ay j + Az k

Relation Between Magnitude and Components

  • The magnitude A of a vector A = Ax i + Ay j + Az k can be found using the Pythagorean theorem:

Conclusion

  • 2D Vectors: A vector in 2D is represented as A = Ax i + Ay j, where Ax = A * cos(Œ∏) and Ay = A * sin(Œ∏).
  • 3D Vectors: A vector in 3D is represented as A = Ax i + Ay j + Az k, where Ax = A * cos(Œ±), Ay = A * cos(Œ≤), and Az = A * cos(Œ≥).
