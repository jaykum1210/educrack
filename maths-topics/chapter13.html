Chapter 13 - Statistics

1. Mean (Average)

Definition

  • The mean (also called the average) is a measure of central tendency in statistics. It is used to summarize a set of numbers by finding a central or typical value.

Formula

  • For a given set of n numbers, the mean is calculated as:
  • ∑ i=1 n xi n

    Where xi are the individual values in the data set and n is the total number of values in the data set.

Types of Mean

  1. Arithmetic Mean

    • The sum of all numbers divided by the number of values.
    • Example: For the set {2, 4, 6, 8}, the mean is:

      • Mean = (2 + 4 + 6 + 8) / 4 = 5

  2. Weighted Mean

    • When different values have different levels of importance or frequency.
    • Formula: ∑ wi xi ∑ wi

Properties of Mean

  • Uniqueness: There is only one mean for any given set of data.
  • Sensitivity: The mean is affected by extreme values (outliers) in the data.
  • Balance Point: The mean is the point where the data set would balance if plotted on a number line.

Applications

  • Used in various fields like economics, education, and science to analyze data.
  • Helps in understanding the distribution of data and making comparisons between data sets.

Example

  • For the data set {5, 10, 15, 20}, the mean is:

    • Mean = (5 + 10 + 15 + 20) / 4 = 50 / 4 = 12.5

2. Median

Definition

  • The median is a measure of central tendency, which represents the middle value of a data set when it is ordered in ascending or descending order.

Steps to Calculate the Median

  • Arrange the data: Arrange the data set in ascending or descending order.
  • Find the middle value:
    • For an odd number of data points: The median is the middle value.
    • For an even number of data points: The median is the average of the two middle values.

Formula

  • Odd number of data points: x n+1 2

    Where n is the total number of values in the data set.

  • Even number of data points: x n/2 2 + x n/2+1 2

    Where n is the total number of values in the data set, and xn/2 and xn/2+1 are the middle two values.

Properties of the Median

  • Insensitive to Outliers: Unlike the mean, the median is not affected by extremely large or small values (outliers), making it a better measure of central tendency for skewed data.
  • Middle Value: The median splits the data set into two equal halves—half of the data points are less than the median, and half are greater than the median.

Example

  • For the data set {3, 1, 4, 2, 5}:

    • Arrange in ascending order: {1, 2, 3, 4, 5}. Since there are 5 values (odd), the median is the middle value, which is 3.

  • For the data set {7, 1, 3, 9}:

    • Arrange in ascending order: {1, 3, 7, 9}. Since there are 4 values (even), the median is the average of the two middle values:

    Median = (3 + 7) / 2 = 5

Applications

  • The median is widely used in fields like economics, healthcare, and education, especially when the data includes outliers or is skewed.
  • It is a better indicator of the central tendency for non-symmetrical distributions.

3. Mode

Definition

  • The mode is a measure of central tendency that represents the value that appears most frequently in a data set. It can be used for both numerical and categorical data.

Steps to Find the Mode

  • Arrange the data: (optional, but can help in identifying the mode).
  • Identify the most frequent value(s): The mode is the value that occurs most frequently in the data set.
  • If one value appears most frequently, it is unimodal.
  • If two values appear with the same highest frequency, the data set is bimodal.
  • If more than two values appear with the same highest frequency, the data set is multimodal.
  • If no value repeats, the data has no mode.

Example

  • For the data set {2, 3, 4, 4, 5, 6, 6, 6, 7}:

    • The number 6 appears three times, more than any other value. Hence, the mode is 6.

  • For the data set {1, 2, 2, 3, 3, 4}:

    • Both 2 and 3 appear twice, and no other number repeats more. This is a bimodal data set with modes 2 and 3.

  • For the data set {1, 2, 3, 4, 5}:

    • All values appear only once. Hence, there is no mode in this case.

Properties of the Mode

  • Can Have More Than One Mode: A data set can be unimodal, bimodal, or multimodal, depending on the frequency of the values.
  • Applicable to Categorical Data: The mode is the only measure of central tendency that can be used with nominal data (data that consists of categories without inherent order).
  • Insensitive to Outliers: The mode is not affected by extreme values, unlike the mean.

Applications

  • In Business: The mode can represent the most popular product or service based on frequency of sales.
  • In Medicine: It can be used to identify the most common symptoms in a population.
  • In Education: Mode can indicate the most frequent grade or score in an exam.

4. Variance

Definition

  • Variance is a statistical measure that represents the spread or dispersion of a set of values. It quantifies how far each number in the data set is from the mean (average) and, therefore, from every other number in the data set. A higher variance indicates that the numbers are more spread out, while a lower variance suggests that the numbers are closer to the mean.

Formula

  • For a population:
    • σ 2 = ∑ xi - μ n

    Where σ² is the population variance, xᵢ are the data points, μ is the population mean, and n is the number of data points.

  • For a sample:
    • s 2 = ∑ xi - x̅ n-1

    Where s² is the sample variance, xᵢ are the data points, x̅ is the sample mean, and n is the number of data points in the sample.

Steps to Calculate Variance

  • Find the Mean: First, calculate the mean (average) of the data set.
  • Subtract the Mean: Subtract the mean from each data point.
  • Square the Differences: Square each of the differences obtained in step 2.
  • Find the Average of Squared Differences:
    • For population variance, divide the sum of squared differences by n.
    • For sample variance, divide the sum of squared differences by n-1.

Properties of Variance

  • Non-negative: Variance is always zero or positive because it involves squaring differences (negative values become positive).
  • Units: The units of variance are the square of the units of the data. For example, if the data is in meters, the variance will be in square meters.
  • Sensitive to Outliers: Variance is sensitive to extreme values (outliers) because large differences from the mean are squared, leading to a larger variance.

Example

  • For the data set {3, 5, 7}:
    • Mean: μ = 3 + 5 + 7 3 = 5
    • Differences from the mean: (3 - 5) = -2, (5 - 5) = 0, (7 - 5) = 2
    • Squared differences: (-2)² = 4, 0² = 0, 2² = 4
    • Sum of squared differences: 4 + 0 + 4 = 8
    • Variance: σ² = 8/3 = 2.67 (for population variance)
  • For a sample (with 3 data points), the sample variance would be:
    • Variance: s² = 8/(3-1) = 8/2 = 4

Applications of Variance

  • In Finance: Variance is used to measure the risk or volatility of an investment's returns.
  • In Quality Control: It helps in assessing the consistency or variation in manufacturing processes.
  • In Data Analysis: It is used to understand the spread of data and the reliability of the mean.

5. Standard Deviation

Definition

  • Standard Deviation is a measure of the amount of variation or dispersion of a set of values. It is the square root of the variance and is used to describe how much individual data points deviate from the mean. Unlike variance, which is in squared units, the standard deviation is in the same units as the original data, making it easier to interpret.

Formula

  • Population Standard Deviation:
    • σ = ∑ xi - μ n

    Where σ is the population standard deviation, xᵢ are the data points, μ is the population mean, and n is the number of data points in the population.

  • Sample Standard Deviation:
    • s = ∑ xi - x̅ n-1

    Where s is the sample standard deviation, xᵢ are the data points, x̅ is the sample mean, and n is the number of data points in the sample.

Steps to Calculate Standard Deviation

  • Find the Mean: Calculate the mean of the data set.
  • Subtract the Mean: Subtract the mean from each data point to find the deviation from the mean.
  • Square the Differences: Square each of the deviations.
  • Find the Average of Squared Differences:
    • For population standard deviation, divide the sum of squared differences by n.
    • For sample standard deviation, divide the sum of squared differences by n-1.
  • Square Root: Take the square root of the variance to get the standard deviation.

Properties of Standard Deviation

  • Non-negative: The standard deviation is always non-negative because it involves taking the square root of a squared quantity.
  • Same Units as Data: Unlike variance, which has squared units, the standard deviation is in the same units as the original data, making it easier to interpret.
  • Sensitive to Outliers: Like variance, the standard deviation is sensitive to extreme values, or outliers, because it involves squaring deviations.
  • Informs on Spread: A larger standard deviation means the data points are spread out more widely from the mean, while a smaller standard deviation indicates the data points are closer to the mean.

Example

  • For the data set {2, 4, 6, 8}:
    • Mean: μ = 2 + 4 + 6 + 8 4 = 5
    • Deviations from the mean: (2 - 5) = -3, (4 - 5) = -1, (6 - 5) = 1, (8 - 5) = 3
    • Squared deviations: (-3)² = 9, (-1)² = 1, (1)² = 1, (3)² = 9
    • Sum of squared deviations: 9 + 1 + 1 + 9 = 20
    • Variance (Population): σ² = 20/4 = 5
    • Standard Deviation (Population): σ = √5 ≈ 2.24
  • For a sample (with 4 data points), the sample standard deviation would be:
    • Variance: s² = 20/(4-1) = 20/3 ≈ 6.67
    • Standard Deviation (Sample): s = √6.67 ≈ 2.58

Applications of Standard Deviation

  • In Finance: To measure the volatility or risk of an investment.
  • In Quality Control: To assess the consistency or precision of manufacturing processes.
  • In Education: To measure how spread out students' scores are from the average score.
